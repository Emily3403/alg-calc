Analytischer Geometrie Rechner von Emily Seebeck

PC:

_____________________________________________________________________________________________________
Der Rechner ist in Python geschrieben, benötigt also python.
Die neuste Version von Python lässt sich hier finden:
https://www.python.org/downloads/
(Aufpassen auf die Plattform)

Ansonsten funktioniert die Installation nicht.

_____________________________________________________________________________________________________
Alle Datein aus Google Drive in ein Verzeichnis herunterladen
_____________________________________________________________________________________________________
Jetzt funktioniert der Rechner bereits. Für die beste Erfahrung würde ich allerdings noch die
Schriftart "DejaVu Sans Mono Unifont" installieren. 

--> Schriftart im Ordner "Fonts" installieren. 
--> Rechtsclick auf die obere zeile (Wo auch das X ist)
--> Eigenschaften
--> Schriftart
--> Bei Schriftart "DejaVu Sans Mono Unifont" auswählen
--> OK

Nun Funktioniert alles und der Rechner lässt sich benutzen
_____________________________________________________________________________________________________

Android:

_____________________________________________________________________________________________________
Lade die alle .py Datein in diesem Ordner herunter (alle außer Installation und Readme)
Installiere eine Python3 Konsole
Ich empfehle hier die App QPython3, allerdings sollte es auch mit jeder anderen App funktionieren

Python Konsole öffnen

Jetzt den Knopf Programs (QPython Startbildschirm) anclicken

Du siehst oben einen Pfad, wo die Skripte gespeichert werden
Dieser ist Standardmäßig unter "Interner Speicher\qpython\scripts3"
Hier müssen alle .py Datein aus diesem Ordner (Analytischer Geometrie Rechner) verschoben werden

Nun die Datei Main.py mit QPython ausführen
